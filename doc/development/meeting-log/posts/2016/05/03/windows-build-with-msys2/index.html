In order to build curie on Windows, we need to build the various Perl dependencies. These Perl dependencies in turn require native libaries such as Gtk+ 3 and Glib. Using the MSYS2 package manager, we can install the native libraries and link against them.

The following steps show how to install the dependencies along with how I ended up debugging the build. The following will be assuming 64-bit Windows (x86-64 architecture) throughout.

Building locally

  1. Install MSYS2. This is a straightforward install into the C:\msys64 directory.

  2. Start the "MinGW-w64 Win64 Shell" from the Start Menu.

  3. Use the pacman manager to update the package database and install the build toolchain:

[[[!format shell pacman -Syu pacman -S --needed mingw-w64-x86_64-toolchain autoconf automake libtool make patch mingw-w64-x86_64-libtool ]]]
